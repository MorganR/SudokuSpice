Declaration
public SquareTracker(Puzzle puzzle, PossibleValues possibleValues, ISudokuRuleKeeper ruleKeeper, ISudokuHeuristic heuristic = null)
Parameters
Type Name Description
Puzzle puzzle

The puzzle to track.
PossibleValues possibleValues

A possible values tracker for the given puzzle.
ISudokuRuleKeeper ruleKeeper

The rule keeper to satisfy when modifying this puzzle.
ISudokuHeuristic heuristic

A heuristic to use to solve this puzzle efficiently. Can be set to null to skip using heuristics.

Note that only one heuristic can be provided. To use multiple heuristics, create a wrapper heuristic like StandardHeuristic.
