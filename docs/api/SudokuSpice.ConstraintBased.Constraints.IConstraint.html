TryConstrain(IReadOnlyPuzzle, ExactCoverMatrix)

Adds necessary ConstraintHeaders and links to the given matrix in order to solve the given puzzle according to this constraint. The details here are implementation-specific.

Declaration
bool TryConstrain(IReadOnlyPuzzle puzzle, ExactCoverMatrix matrix)
Parameters
Type Name Description
IReadOnlyPuzzle puzzle

The puzzle to solve.
ExactCoverMatrix matrix

The exact-cover matrix to constrain.
Returns
Type Description
System.Boolean

False if the constraint could not be satisfied by the given puzzle, else true.
Remarks

This should skip adding ConstraintHeaders that are already satisfied by the given puzzle. Instead, it should drop the corresponding PossibleSquareValues that would have been included in these headers.

Note: See ExactCoverMatrix to understand how the matrix works.
