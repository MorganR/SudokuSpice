Class UniqueInColumnHeuristic

Checks for any squares that are the unique provider of a given possible value within a column. Sets the possible values for those squares to just their unique value.

Remarks

For example, if a column had three unset squares with possible values: A: [1, 2], B: [1, 2], and C: [1, 2, 3], then this would set C's possible values to [3].
