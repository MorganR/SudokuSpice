PuzzleWithPossibleValues(Nullable<Int32>[,])

Constructs a new puzzle backed by the given array.

The puzzle is backed directly by this array (i.e. modifying the array modifies the puzzle, and vice-versa). If this is not what you want, see CopyFrom(Nullable<Int32>[,]). Note that all future modifications should be done through this puzzle object, else this will be in an incorrect state.
